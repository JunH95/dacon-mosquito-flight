# Dacon 모기 비행 예측 - Phase 5 (모델 고도화 계획서)

> **Agent Dacon Expert 님의 리뷰 결과입니다.**

현재 완성된 코드는 가장 기본적인 "Baseline(뼈대)"입니다. 모기가 날아가는 단순 좌표(`x, y, z`)만 LSTM 모델에 입력으로 넣고 있어 예측 점수가 매우 낮게(0.02점대) 나오는 것이 당연합니다. 상위권에 도달하기 위한 핵심 업그레이드 전략을 제안합니다.

## 🚀 제안하는 업그레이드 방안 (Phase 5)

### 1. 물리적 파생 변수 딥러닝 주입 (Feature Injection) - **현재 진행 중**
- **현재:** LSTM 입력으로 `x`, `y`, `z` 3개만 사용.
- **변경:** 이미 우리가 만들어 둔 `src/features.py`를 활용하여 각 타임스텝별 **속도(Velocity)** 와 **가속도(Acceleration)** 를 계산하여 LSTM에 함께 주입합니다. (입력 차원이 3차원에서 5차원으로 확장됨)
- **효과:** 모기가 갑자기 꺾거나 가속하는 물리적 특징을 딥러닝 모델이 직접 학습할 수 있어 즉각적인 점수 상승이 예상됩니다.

### 2. K-Fold Cross Validation (교차 검증 및 앙상블) - **대기 중**
- **현재:** Train 데이터를 100% 통째로 학습하고 바로 Test 데이터를 추론.
- **변경:** 훈련 데이터를 5개 조로 쪼개어(5-Fold) 5개의 각기 다른 모델을 학습시키고, 마지막 추론 시 5개 모델의 예측 좌표의 평균(Average Ensemble)을 제출.
- **효과:** Dacon 대회에서 점수를 안정적으로 올릴 수 있는 필수 테크닉입니다.

### 3. LightGBM 모델 추가 개발 및 앙상블 - **대기 중**
- **변경:** 딥러닝(LSTM)과 구조가 아예 다른 머신러닝(LightGBM) 모델을 개발(`src/models/baseline_lgb.py`)하여 결과를 합칩니다.
